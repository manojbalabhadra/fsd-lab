Working of Hyperlinks in HTML

The <a> tag is used to create hyperlinks in HTML. It uses the href attribute to define the link destination and the target attribute to define how the link opens.

Examples:

Open Google (Same Tab)

Open Wikipedia (New Tab)

Open GitHub (Parent Frame)

Open W3Schools (Top Frame)

Go to Section 2 (Internal Link)

Section 2

This is Section 2, reached using an internal hyperlink.
